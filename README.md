# Watermark

> Watermark component for React.js

## Usage Guide

1. Install

   ```bash
   npm install @RunFridge/watermark
   ```

2. Import and use it!

   ```tsx
   import Watermark from "@RunFridge/watermark";

   const App = () => {
     return <Watermark isVisible={true} text="DEVELOPMENT" />;
   };
   ```
